# Niriha: a minimal, fast Wayland setup with Niri, Waybar, and Fuzzel/Vicinae.

A clean, lightweight configuration focused on speed, simplicity, and a distraction‑free workflow. Everything is fine‑tuned for consistency across the Wayland environment built on Arch Linux with a focus on simplicity and clean configuration.

## Tools Used

* mako - Notification daemon for minimal, unobtrusive alerts
* waybar - Highly customizable status bar for Wayland
* btop - Resource monitor to track system performance
* fuzzel - Lightweight application launcher (similar to dmenu)
* vicinae - Quick search and workflow enhancer for app launching and actions
* ghostty - Terminal multiplexer for background session management
* starship - Minimal, fast, and highly configurable shell prompt
